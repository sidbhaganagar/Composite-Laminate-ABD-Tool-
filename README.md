# Composite-Laminate-ABD-Tool-
Open‑source ABD matrix calculator for composite laminates using Classical Lamination Theory (CLT). Computes A, B, and D stiffness matrices from user‑defined ply orientations, thicknesses, and material properties. Useful for materials science, structural analysis, composite design, and engineering education.
The calculator is provided as an easy‑to‑use Excel spreadsheet, making it accessible for engineers, researchers, and students working in:

Composite materials

Structural mechanics

Materials science

Reliability engineering

Energy‑system components

Aerospace and mechanical design


Features
Computes A, B, and D matrices for any laminate stacking sequence

Supports arbitrary ply angles

Allows custom material properties for each ply

Automatically calculates transformed reduced stiffness matrices

Provides intermediate calculations for transparency and verification

No programming required — fully Excel‑based


How to Use
Open the Excel file in this repository.

Enter ply material properties (E₁, E₂, G₁₂, ν₁₂).

Enter ply thicknesses and fiber orientations.

The spreadsheet automatically computes:

Q̅ matrices

A, B, D stiffness matrices

Laminate coupling behavior

Applications
This ABD calculator is useful for:

Designing composite laminates

Evaluating stiffness and coupling behavior

Teaching CLT concepts

Supporting structural and materials‑engineering research

Modeling components used in energy‑system infrastructure
